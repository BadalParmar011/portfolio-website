    Blog Web-App

    Visual Question Answering

    An attention-based classification model that aims at generating an answer for a given input image.
    1. Incorporated Convolution Neural Networks (CNN) for extracting image features and Long Short Term Memory for extracting question embeddings.
    2. Tested the model on the COCO dataset, abstract scenes images, and got 69% overall accuracy on the VQA evaluation metric.

    Video Summarizer

    A Seq2Seq model that generates a short summary of the given input video.
    1. Incorporated CNN to detect and classify objects in the video frames and Long Short Term Memory for generating a summary.
    2. Evaluated the model on MSVD (Microsoft Video Description Corpus) dataset; achieved 0.77, 0.71, 0.52 scores respectively on ROGUE, BLEU, METEOR evaluation metrics.

    Head Counting System

    A system that calculates the attendance of the class from a panoramic image of a live classroom.
    1. Used Singular Value Decomposition for image compression; applied various image processing techniques and morphological operations to detect the number of heads.
